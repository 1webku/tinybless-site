Cara Membuat Link WhatsApp untuk Bio Link

WhatsApp sering digunakan untuk komunikasi dengan pelanggan atau followers. Dengan menggunakan link WhatsApp, orang dapat langsung membuka chat tanpa perlu menyimpan nomor terlebih dahulu.

Link WhatsApp ini biasanya dimasukkan ke halaman bio link yang berada di bio Instagram atau TikTok.

1. Format Link WhatsApp

Format dasar link WhatsApp adalah:

https://wa.me/NOMOR

Contoh:

https://wa.me/6283124254787

Ketika link tersebut dibuka, WhatsApp akan langsung membuka chat ke nomor tersebut.

2. Link WhatsApp dengan Pesan Otomatis

Kamu juga bisa menambahkan pesan otomatis menggunakan format berikut:

https://wa.me/NOMOR?text=PESAN

Contoh:

https://wa.me/6283124254787?text=Halo%20saya%20ingin%20order

Teks "%20" digunakan sebagai pengganti spasi.

3. Memasukkan Link WhatsApp ke Bio Link

Jika kamu memiliki banyak link seperti Instagram, Shopee, atau YouTube, kamu bisa menggabungkannya dalam satu halaman bio link.

Contoh isi halaman bio link:

WhatsApp
Shopee
Instagram
YouTube

Tombol WhatsApp akan mengarah ke link seperti:

https://wa.me/6283124254787

4. Letak Bio Link di Instagram atau TikTok

Setelah halaman bio link selesai dibuat, kamu bisa menaruhnya di bio Instagram atau TikTok.

Alurnya biasanya seperti ini:

Instagram / TikTok
↓
Bio Link
↓
WhatsApp

5. Manfaat Menggunakan Bio Link

Bio link memudahkan followers menemukan semua link penting kamu dalam satu halaman.

Kamu bisa menampilkan:

WhatsApp
Toko online
YouTube
Website
